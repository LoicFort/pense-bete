# Terminal

- ouvrir = ctrl+alt T
  
- ctrl S = save

- logiciel . permet d'ouvrir le dossier dans lequel on se trouve. Pour VScode par exemple
  
- ~ = racine + user ex: /home/student

- cd = create directory

- mv = moove ex: mv text.txt ~/Documents // ou renommer ex: mv text.txt text2.txt

- mkdir = make directory 

- rmdir = supprimer dossier vide 

- rm = permet de supprmier fichier ex rm test.txt

- rm - r = attention!! = permet de supprimer un répertoire et tout son contenu

# CSS

- ctrl+/ = mettre un commentaire
- ul>li*3 = permet de créer balises UL avec 3 li à l'intérieur 
- mettre balises autour d'un mot surligné = ctrl+$ puis mettre la balise
- ctrl shift i = pour reformater le texte
- on peut attribuer plusieurs attributs dans une balise en mettant un espace : ex <p class="1 2">
- En hexa quand les 3 paires de caractères se répètent on peut abréger. ex: FF9900 = F90
- `Div` est de type **block**, ``span`` de type **inline**
- reset css: permet de virer tous les styles, pour commencer un nouveau projet par exemple. 
 on peut créer un fichier reset et le charger dans son html. A positionner avant le lien de chargement de notre stylesheet.
